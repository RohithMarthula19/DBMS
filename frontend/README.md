# Frontend — Double-Entry Banking System

React + Vite single-page app that consumes the backend REST API.

```
src/
  api/          # API client functions
  components/   # reusable UI components
  pages/        # route-level views
```

## Setup

```bash
npm install
npm run dev
```

By default this expects the backend to be running locally (see the root [README](../README.md) for backend setup).
